# MyappDiesel

Open Road Diesel Repair website project.

## Open in PyCharm

1. Open PyCharm.
2. Select **File > Open**.
3. Choose this folder:

   `C:\Users\aanikitenko\Documents\MyappDiesel`

## Run locally

```powershell
npm install
npm run dev
```

The local website runs at:

`http://127.0.0.1:3001`
